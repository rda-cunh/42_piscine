# 42 Piscine — C Piscine Deliverables

## Overview
This repository contains the work completed during the 42 School C Piscine in 2023. The Piscine is a 26-day intensive learning experience designed to introduce students to C programming, Unix tools, and the 42 pedagogy of learning by doing.

This was my first real coding experience. For me, the value of the Piscine was not in the volume of code produced, but in what I learned and what I was able to achieve in a short time.

## Structure
- `C00/` — basic output and character exercises
- `C01/` — pointer handling and simple operations
- `C02/` — string handling and character tests
- `C03/` — string functions and concatenation
- `C04/` — more utilities including print and conversion functions
- `C05/` — algorithms and recursive exercises
- `C06/` — command-line parameter handling and sorting
- `Shell00/`, `Shell01/` — shell scripting exercises related to the 42 Shell module

## Notes
- I was unable to deliver `C05` successfully because of an issue with a 42 header that prevented the moulinette from passing. The code itself was attempted, but it failed the required validation.

## Reflection
Looking back, I was not very strong at the beginning, but the Piscine was the start of a great learning journey. The early mistakes and the steep learning curve were part of the experience.

The main takeaway is that the Piscine is about rapid growth: learning the basics of C, understanding how to think like a programmer, and building confidence with a new technical discipline.

## Usage
